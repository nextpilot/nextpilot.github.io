# NextPilot 飞控系统用户手册

NextPilot 一款功能丰富、高度扩展、稳定可靠的先进自动驾驶仪，**基于国产RT-Thread操作系统，并支持全国产化硬件定制**。它支持多旋翼、固定翼和垂起无人机，**支持Simulink基于模型设计**，可便捷地应用于教育、研究和工业等领域。

本指南涵盖从装机、配置、飞行操作和数据分析的各个环节。

## NextPilot 特色

- 支持舰载移动平台自主起降
- 支持基于视觉的辅助起降
- 支持高精度精准空投
- 支持集群多机编队控制
- 搭载高精度工业级IMU，**无需外部减震**
- 地面控制站集成虚拟飞行仿真
- 地面控制站具备日志记录和回放
- 地面控制站支持3D航线规划

## 体验 NextPilot

不需要硬件，NextPilot-Ground-Control 地面控制站集成了[虚拟飞行仿真](03-飞行仿真/01-软件在环仿真.md)功能。在地面控制站中创建一个虚拟飞机链接，即可立即开始飞行。
