# 机架说明

​  无人机执行器一般包括舵机、电调电机、发动机等，驱动信号一般为PWM。飞控提供了共计16路PWM输出通道，根据不同机型，需要对应连接。

​  关于不同机型具体对应的机架在[机架设置](../02-飞控设置/01-基础设置.md#机架设置)章节。
