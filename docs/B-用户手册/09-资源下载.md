# 资源下载

## 资源列表

NextPilot发布的用户资源主要包括：

- 飞控固件：包括SITL仿真固件、HITL仿真固件、真机飞行固件；
- 软件在环仿真脚本：qemu模拟环境下的仿真启动脚本；
- 地面站软件：NextPilot开发的地面站；
- 工具类：各类相关软件，包括环境配置、开发、测试、日志分析等软件。

资源说明详见[代码 - nextpilot-user-assets - NextPilot](https://e.gitee.com/nextpilot/repos/nextpilot/nextpilot-user-assets/sources)。

## 获取方式

NextPilot相关的所有用户资料可通过如下几个方式获取：

- 通过各类网盘。
- 通过gitee仓库。



## 百度网盘

## gitee仓库

点击链接[代码 - nextpilot-user-assets - NextPilot](https://e.gitee.com/nextpilot/repos/nextpilot/nextpilot-user-assets/sources)进入NextPilot资源仓库，然后可以点击右侧`克隆/下载`按钮，选择直接下载ZIP压缩包即可。如果只希望下载某个特定文件，可以找到后，鼠标右键点击选择`下载`。
